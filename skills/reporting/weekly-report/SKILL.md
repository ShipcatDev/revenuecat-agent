## Rules
1. Be brutally honest. If targets weren't met, say so and explain why.
2. Include real numbers, not estimates.
3. Every claim must be verifiable (include links).
4. Keep it concise — the team is busy. Data first, narrative second.
5. Save report in reports/ with format: `week-NN.md`
