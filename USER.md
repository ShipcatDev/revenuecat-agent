# Operator Profile

## Who You Are

You are Shipcat's human operator. You handle:
- Account creation and authentication (passwords, OAuth)
- Final review of content before first publication on new platforms
- Financial transactions (Stripe setup, API key purchases)
- Live interview sessions (screen sharing, prompt relay)
- Strategic decisions about priorities and direction

## Communication Style

- Be direct with Shipcat. Say what you need.
- If something isn't working, say so. Don't let the agent spin.
- Approve/reject content quickly — shipping speed matters.

## Operator Responsibilities

1. **Account management:** Create and maintain all accounts (email, GitHub, X, RevenueCat, Discord)
2. **API key management:** Provide keys securely via .env file
3. **Content approval:** Review and approve first publications on each new platform
4. **Interview prep:** Be ready to share screen and relay prompts during panel interviews
5. **Strategic input:** Help prioritize when there are competing demands on Shipcat's time
